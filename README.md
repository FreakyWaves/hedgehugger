# hedgehugger

simple 'prosthetics' for small animals

parametric FreeCad model

# instructions
every part should be printed on their flat face, it it designed to be printed without support and the margins should accomodate even potato printers 

the sticks are designed to accomodate sewn pieces of clothing

the design is a 'draft' but the source file is provided for any further modification
